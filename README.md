# Vendor-performance-Analysis
This project focuses on analyzing vendor-level sales and purchase data to uncover key business insights related to revenue contribution, profitability, and vendor performance. The goal is to identify high-performing vendors, detect inefficiencies, and support data-driven decision-making.

Tools & Technologies:
Python (Pandas, NumPy)
Data Visualization: Matplotlib, Seaborn
Jupyter Notebook

Business Objectives:
Identify top vendors contributing to total purchases
Analyze vendor-wise profitability and margins
Detect high-revenue but low-profit vendors
Evaluate pricing trends across order sizes
Assess vendor dependency using Pareto analysis (80/20 rule)

Key Insights:
A small number of vendors contribute the majority of total purchases (Pareto principle)
Some vendors generate high revenue but low profit → potential cost inefficiencies
Pricing varies significantly across order sizes, indicating negotiation opportunities
Vendor dependency risk identified due to concentration of purchases

Conclusion:
This analysis helps businesses optimize vendor relationships, improve profitability, and reduce operational risks by leveraging data-driven insights.
